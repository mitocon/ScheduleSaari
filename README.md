# ScheduleSaari
## 予定登録の手軽さに特化したスケジュールアプリ
- Googleカレンダーの操作を複雑に感じる
- もっと素早く予定を登録したい
- 機能は少なくていい

そんな方向けのスケジュールアプリです。
めんどくさがり屋のためのスケジューラーです。

## 使い方
以下のように入力すると6月3日の予定に追加されます。  
```
6/3 9:00 駅に集合
```
6月を06、3日を03と表現しても登録可能です。  
以下の4例はすべて同じ結果になります。
```
6/3 9:00 駅に集合
06/03 9:00 駅に集合
06/3 9:00 駅に集合
6/03 9:00 駅に集合
```

彼女と同様、"登録"という作業が苦手なために、
